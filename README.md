# Coinvera Exchange Desk

Modern React + TypeScript frontend for Coinvera's USDT/INR exchange desk.

## Stack

- React 19
- TypeScript
- Vite
- Component-based pages
- Typed local storage services
- TradingView widget integration
- Responsive fintech UI

## Run Locally

```powershell
npm run dev -- --port 5174
```

Open:

```text
http://127.0.0.1:5174/
```

## Pages

- `/` market dashboard and login/signup
- `/buy` buy USDT flow
- `/sell` sell USDT flow
- `/orders` customer order tracking
- `/admin` admin rates, payment setup, and order controls

## Production Build

```powershell
npm run build
```
